## 🚀 快速开始

### 前置准备
1. 安装 **DevEco Studio**
2. 注册 **华为开发者账号**
3. 准备 **HarmonyOS 设备** 或 **模拟器**

### 后端服务启动
```bash
# 进入后端目录
cd docs/py

# 安装依赖
pip install -r requirements.txt

# 启动服务
python -m uvicorn backend.main:app --host 0.0.0.0 --port 6090 --reload
```

### 鸿蒙App 配置
1. 打开 `harmony_app` 目录
2. 修改 `entry/src/main/ets/utils/Constants.ets` 中的后端地址：
   ```typescript
   // 模拟器访问主机
   static readonly BASE_URL: string = 'http://10.0.2.2:6090';
   
   // 真机调试（替换为电脑实际IP）
   // static readonly BASE_URL: string = 'http://192.168.x.x:6090';
   ```

### 测试账号
| 角色 | 账号 | 密码 |
|------|------|------|
| 平台管理员 | platform_admin | Platform@2026 |
| 租户管理员 | tenant_admin | Tenant@2026 |
