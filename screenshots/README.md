# Screenshots Guide / 截图指南

本文档说明如何为项目生成和添加截图。

## 需要的截图

### 1. 登录页面 (login_v2.html)
- 访问 http://localhost:6090/login
- 截取完整页面
- 文件名: login-page.png

### 2. 管理后台 (admin_v2.html)
- 访问 http://localhost:6090/admin
- 截取租户管理面板
- 文件名: admin-dashboard.png

### 3. 租户门户 (tenant_v2.html)
- 访问 http://localhost:6090/tenant
- 截取智能体管理界面
- 文件名: tenant-portal.png

### 4. 数据分析 (analytics_v2.html)
- 访问 http://localhost:6090/analytics
- 截取数据分析面板
- 文件名: analytics-dashboard.png

### 5. 平台分析 (platform_analytics_v2.html)
- 截取跨租户分析面板
- 文件名: platform-analytics.png

### 6. 鸿蒙 App
- 在 DevEco Studio 模拟器中运行
- 截取对话界面
- 文件名: harmonyos-chat.png
- 截取智能体选择界面
- 文件名: harmonyos-agents.png

## 截图规范

- 分辨率: 至少 1920x1080
- 格式: PNG
- 命名: 小写字母 + 连字符
- 位置: screenshots/ 目录
- README 中引用: ![描述](screenshots/文件名.png)
