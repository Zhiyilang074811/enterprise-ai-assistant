# 华为应用市场上架检查清单

## 上架前检查

### 基本信息
- [ ] 华为开发者账号已注册并完成认证
- [ ] 应用名称已确定（中文 + 英文）
- [ ] 应用分类已选择（办公/效率工具）
- [ ] 应用图标已准备（108x108 + 512x512 PNG）
- [ ] 应用截图已准备（至少 2 张，推荐 5 张 1080x2340）
- [ ] 隐私政策文档已准备
- [ ] 应用描述已撰写（简短 + 完整）

### 技术检查
- [ ] DevEco Studio 已安装（4.0+）
- [ ] HarmonyOS SDK NEXT 已安装
- [ ] 签名证书已配置（.cer + .p7b）
- [ ] 应用能在模拟器上正常运行
- [ ] 应用能在真机上正常运行
- [ ] 所有 API 端点已配置正确的后端地址
- [ ] 错误处理已完善
- [ ] 网络请求超时已设置

### 内容检查
- [ ] 应用描述准确反映功能
- [ ] 无违规内容
- [ ] 无敏感权限滥用
- [ ] 隐私政策合规
- [ ] 关键词已优化

### 提交前检查
- [ ] 版本号已递增
- [ ] 更新日志已填写
- [ ] 应用图标无瑕疵
- [ ] 截图清晰美观
- [ ] 联系方式已填写

## 上架后运营

- [ ] 及时回复用户反馈
- [ ] 监控应用性能数据
- [ ] 规划下一个版本
