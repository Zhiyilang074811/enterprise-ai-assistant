# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-17

### Added
- Multi-tenant enterprise AI assistant platform
- RAG pipeline with hybrid retrieval (BM25 + TF-IDF + Dense) and reranking
- HarmonyOS native mobile application (ArkTS + ArkUI)
- Platform-level and tenant-level analytics dashboards
- Knowledge base with document processing and crawling
- Workflow engine with configurable business processes
- Content guardrails and security controls (input + output)
- Double-backend deployment mode
- JWT authentication with RBAC
- Vector database integration (Qdrant / Milvus)
- Pluggable LLM backend (DashScope, OpenAI, domestic models)
- Semantic cache with similarity detection
- Balance management and rate limiting
- Evaluation framework for retrieval quality
