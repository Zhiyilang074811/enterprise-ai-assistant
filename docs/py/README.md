# 服务商版

这是根据当前母版源码自动裁剪出来的 服务商版 交付包。

## 版本定位

- 版本名称：服务商版
- 部署模式：`double_backend`
- 适用对象：服务商、多客户托管方、SaaS 运营方
- 版本简介：双后台版本，适合服务商、多客户托管和 SaaS 运营。

## 默认入口

- 平台后台：`/admin`
- 租户后台：`/tenant`
- 登录入口：`/login`
- 聊天入口：`/chat`

## 交付重点

- 平台后台 + 租户后台双后台交付
- 支持多企业客户统一管理与分租户配置
- 统一运营、多智能体装配、工作流交付与客户隔离并存
