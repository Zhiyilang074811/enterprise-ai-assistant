Enterprise RAG

企业知识库 Agent

将制度、SOP、产品文档、项目资料与实时数据接入统一知识大脑，提供检索、问答、模型调度、采集调度与审计留痕的一体化企业 Agent 平台。

语义检索

支持关键词、向量与混合检索，结合 Embedding 与 Rerank 做多文档召回、精排与答案生成。

企业隔离

每个租户拥有独立的 Prompt、知识库、模型配置、采集任务、成员账号与日志空间。

安全护栏

支持输入拦截、输出脱敏、请求日志、审计事件留痕与风险内容追踪。

自动采集

支持网站、API 与数据库采集规则配置，按计划自动执行并回写知识库。

模型轮换与 Key 池

支持多 Key 自动轮换、主备模型降级与提供方切换，提升高并发和异常场景下的稳定性。

调度与可观测

内置检索解释、对话审计、评测实验与调度中心，方便排查效果、追踪链路与验证配置。
