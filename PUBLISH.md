## Press Kit

### Project Name
Enterprise Multi-Tenant AI Assistant Platform (企业级多租户智能问答流程助手平台)

### Tagline
Built for enterprises. Powered by AI. Open Source.

### Short Description (140 chars)
Open-source multi-tenant AI assistant with RAG, LangChain agents, HarmonyOS app. Enterprise-ready.

### Key Highlights
- Multi-tenant SaaS architecture with complete data isolation
- RAG pipeline with hybrid retrieval + reranking
- LangGraph agent orchestration
- HarmonyOS native mobile app (ArkTS)
- 40+ REST API endpoints
- Enterprise security: dual guardrails, audit logging, rate limiting
