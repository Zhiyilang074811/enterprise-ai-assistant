## 三、打包发布流程

### 1. 签名配置
1. 打开 DevEco Studio
2. File -> Project Structure -> Signing Configs
3. 导入签名证书 (.cer + .p7b)
4. 填写密钥库信息

### 2. 构建 Hap 包
1. Build -> Build Hap(s)/APP Bundle(s) -> Build Hap(s)
2. 等待构建完成
3. 在 entry/build/outputs/ 中找到签名后的 .hap 文件

### 3. 提交审核
1. 登录华为开发者联盟: https://developer.huawei.com/consumer/cn/service/josp/aggregation.html
2. 进入「我的应用」-> 「应用上架」
3. 创建新应用
4. 上传 .hap 文件
5. 填写应用信息（名称、描述、截图、分类等）
6. 上传隐私政策文档
7. 提交审核

### 4. 审核周期
- 通常 1-3 个工作日
- 审核期间保持联系方式畅通
- 如有问题及时修改

## 四、上架后运营

### 1. 版本管理
- 每次更新递增版本号
- 在更新日志中说明变更内容
- 重大更新提前通知用户

### 2. 用户反馈
- 及时回复应用市场评论
- 收集用户需求
- 定期更新优化

### 3. 数据统计
- 监控下载量、活跃度
- 分析用户行为
- 优化产品体验

## 五、审核注意事项

### 常见拒绝原因
1. 隐私政策不完善
2. 应用描述与实际功能不符
3. 存在明显 Bug 或崩溃
4. 权限申请不合理
5. 包含违规内容

### 规避建议
1. 完善隐私政策，明确数据收集范围
2. 确保应用描述准确反映功能
3. 充分测试，避免崩溃
4. 只申请必要的权限
5. 遵守华为应用市场规范
